Fallout Banners

In 2008, AKQA brought me on to help out with a tight deadline for a banner campaign for the new Fallout 3 video game. The banner ads were already storyboarded before I was brought on, so my role was to bring the static mock-ups to life using Flash. The big challenge of this project was meeting the strict file size requirements for the publishers while still delivering a high quality product.

  • Role: Flash Animation, ActionScript Development
  • Client: Bethesda Software
  • Agency: AKQA
  • Year: 2008
